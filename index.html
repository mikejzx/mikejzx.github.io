Michael Skec’s Webpage

The home of legends

Well, lucky you — you've found my humble webpage out here in the World Wide Web. All you will really find here is a bit of contact information (see near bottom of page), and a “blog” which I may occasionally post to.

The blog section of this site runs on a modified version of Roman Zolotarev's epic script. Big thanks to him for writing that. I'm hosting the actual site using GitHub Pages currently, since of course it's free, and I don't need any server-side code. (Site repository)

There is a small amount of (optional) JavaScript that runs on this specific page; its sole purpose is to make getting my e-mail address easier for humans, and more difficult for robots. If you choose to disable JavaScript you will see the address in a slightly less-readable form.

The content on this site is also mirrored in geminispace! If you too believe that the web today is bloated, and you haven't heard of Gemini, be sure to check it out at the project page!

Nevertheless, thanks for stopping by!
